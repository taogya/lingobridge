# 最初の翻訳を実行する

どこからでも翻訳できます。

- **ドキュメント全体** — ファイルを開いた状態で `Cmd/Ctrl+Alt+L` を押す
  (またはエディタ右上の地球アイコンをクリック)。
- **選択範囲** — テキストを選択して `Cmd/Ctrl+Alt+L` を押す
  (翻訳パネルが開き、選択範囲が入力欄にプリフィルされます)。
- **翻訳パネル** — **入力** ボックスにテキストを貼り付けて Enter。

方向別ショートカット:

| 操作 | ショートカット |
| --- | --- |
| JA → EN | `Cmd/Ctrl+Alt+E` |
| EN → JA | `Cmd/Ctrl+Alt+J` |
| 選択範囲のトークン推定 | `Cmd/Ctrl+Alt+T` |
| 翻訳パネルにフォーカス | `Cmd/Ctrl+Alt+Shift+L` |

これで準備完了です。
