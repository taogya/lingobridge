# Markdown記法とシンボル網羅テスト計画

## カバー対象

### 1. Markdown基本構造要素
- [x] 見出し: `# H1` / `## H2` / `### H3` 〜 `###### H6`
- [x] 段落: 通常テキスト（改行なし）
- [x] 改行: 空行で区切られた複数段落
- [x] 強調: `*italic*`, `**bold**`, `***bold-italic***`
- [x] インラインコード: `` `code` ``
- [x] フェンスコード: ` ```lang ... ``` `
- [x] リスト（unordered）: `- item` / `* item` / `+ item`
- [x] リスト（ordered）: `1. item` / `2. item`
- [x] リスト（ネスト）: `  - nested`
- [x] 引用: `> quote`
- [x] 引用（ネスト）: `> > nested quote`
- [x] テーブル: `| col | col |` / `| --- | --- |`
- [x] 水平線: `---` / `***` / `___`
- [ ] リンク: `[text](url)`
- [ ] 画像: `![alt](url)`
- [ ] 脚注: `[^1]` / `[^1]: definition`
- [ ] 定義リスト: `term\n: definition`
- [ ] HTML混在: `<span>text</span>`

### 2. 文章でよく使われる記号
- [x] 句読点: 。、，
- [x] かっこ: 「」『』()[]{}
- [x] 記号: ・…‥！？
- [x] 矢印: →←↑↓
- [x] 通貨: $€¥
- [ ] 数式: `$...$` または `$$...$$`
- [ ] 特殊文字: ©®™§¶†‡
